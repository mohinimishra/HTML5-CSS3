# CSS Grid-Gap Rlated Properties -

* grid-row-gap

* grid-column-gap

* grid-gap

## grid-row-gap property - 
```css
grid-row-gap : 10px;   /* give space btw grid rows
```

## grid-column-gap property - 
```css
grid-column-gap : 10px;   /* give space btw grid columns
```

## grid-gap property - 
```css
grid-row-gap : 10px , 15px;   /* give space btw grid rows & columns.
```
